B - Modifier et/ou supprimer une vue

 
Il est possible de modifier vos vues personnalisées.
 
1.     Cliquez sur l'onglet VUE  afin d'accéder à la liste des vues disponibles.
2.     Sélectionnez l’action souhaitée en cliquant sur les boutons situés pour chaque vue à droite du nom de la vue
3.     Apportez les modifications nécessaires en suivant le process précédemment défini
4.     Sauvegardez
 
Vous ne pouvez apporter des modifications que sur les vues personnalisées que vous avez créées. Ainsi, vous ne pourrez pas ajouter ou supprimer de champs sur les vues créées par d’autres utilisateurs même si celles-ci sont rendues publiques et donc visibles. Vous ne pourrez également pas supprimer les vues rendues publiques que vous n’avez pas vous-même créées.
Vous pouvez en revanche choisir d’afficher ces vues publiques dans les différents onglets (opportunités, projets…) et d’en faire votre vue par défaut.
 
Les utilisateurs ne pouvant modifier que leurs propres vues, les boutons permettant la modification ne sont donc accessibles que sur leurs vues.
 
Pour mettre à jour une vue, il est également possible de cliquer directement sur les carrés disponibles pour chaque vue dans la liste des vues. Lorsque l’option est sélectionnée, le carré devient bleu.
