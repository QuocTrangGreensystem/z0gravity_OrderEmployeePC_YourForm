D - Tâches

 
Une fois les phases et les jalons créés, vient le temps d'ajouter les tâches à votre projet via l'onglet TACHES.
Les phases définies dans l'onglet PLANNING sont affichées dans cet onglet comme constaté ci-dessous:.
 
Rappel de l'écran "PLANNING":
 
Retranscription des phases dans l'écran tâches:
 
Pour rappel, vous pouvez créer 3 niveaux dans votre GANTT :
- Niveau 1 : Phase
- Niveau 2 : Tâche
- Niveau 3 : Sous tâche
 

Le contenu de D - Tâches

a - L'écran Tâche en détail
b - Créer une tâche
c - Créer une tâche non continue
d - Créer une sous-tâche
e - Supprimer une tâche
f - Déplacer une tâche
g - Afficher les prédécesseurs
h - Filtrer les tâches
i - Vérifier la disponibilité d'une ressource
j - Repositionner un projet dans le temps
k - Commentaires et documents liés aux tâches
l - Autres fonctionnalités sur l’écran dédié aux tâches
m - Geler les données initiales
n - Afficher et modifiez vos tâches sous forme de KANBAN
o - Créer une tâche à partir des vues VIGNETTES et LISTE
