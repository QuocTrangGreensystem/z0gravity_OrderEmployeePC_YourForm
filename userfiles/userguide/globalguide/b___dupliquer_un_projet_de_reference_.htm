b - Dupliquer un projet de référence

 
 Il est également possible de créer un projet par duplication à partir de la vue VIGNETTES ainsi que de la vue LISTE.
 

i. A partir de la vue VIGNETTES

 A partir de la vue « vignettes », cliquez sur le bouton
 
Une fenêtre s’ouvre :
 
 
  • Sélectionnez « créer un projet »
     
  • Cliquez sur « à partir d’un modèle »
     
    3. Sélectionnez le projet à dupliquer dans la liste. Ne figurent dans cette liste que les projets « modèles ». Il convient donc, si vous utilisez régulièrement la duplication de projets, de vous créer une liste de projets modèles dédiée (voir C.)
 
4. Cliquez sur « dupliquer votre projet  »
 
 
5. Renommez le projet selon le formalisme à respecter. Vérifiez / Modifiez également le nom du Chef de Projet. Attention, il n’y a qu’un porteur du projet.
 
 
Le nouveau projet est créé et hérite des caractéristiques du modèle. Il figure bien parmi les projets en opportunité.
 
 
 
 

ii. A partir de la vue LISTE

A partir du menu « liste » la procédure est la même, accessible grâce au bouton
 
 
Le projet nouvellement créé hérite d’un statut « opportunité ».
