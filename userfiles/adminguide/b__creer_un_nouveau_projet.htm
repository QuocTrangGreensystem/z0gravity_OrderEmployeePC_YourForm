B. Créer un nouveau projet

 
 
Pour créer un nouveau projet, cliquez sur l'onglet "opportunité"
 
 
Puis cliquer sur 
 
 
 
Lorsque vous créez un projet, vous avez deux possibilités: dupliquer un projet de référence ou créer un projet vide.
 

  • Dupliquer un projet de référence

     
1.     Sélectionner «  dupliquer le projet »
 
 
 
2.     Sélectionner un projet dans la liste puis cliquer sur "OK", sinon sélectionner  "Abandonner ». Ne figurent dans cette liste que les projets « modèles ». Il convient donc, si vous utilisez régulièrement la duplication de projets, de vous créer une liste de projets modèles dédiée (voir C.)
 
 
3.     Préciser le nom du projet puis sauvegarder
 
 

b.  Créer un projet vide

 
1.     Sélectionner « ne pas dupliquer le projet »
 
 
2.     Préciser le nom du projet, le chef de projet, puis sauvegarder à l’aide du bouton en bas de page
