Statuts d'un projet

 
Un projet peut avoir 3 statuts.
 
 
  • Statuts par défaut : Opportunité/Opportunity
     
    Lorsque que vous créez un projet le statut du projet par défaut est "Opportunity/Opportunité". Avec ce statut, le projet n'est pas visible du plan de charge global de l'entreprise et n'est pas ouvert à la saisie des temps.
     
    Pour changer le statut, sélectionner l'onglet "Detail", changer le statut, sauvegarder le projet.
     
     
  • Passage du statut "Opportunity/Opportunité" vers le statut "In progress/En cours"
     
    Sélectionner l'onglet "Detail/Détail", changer le statut, sauvegarder.
     
     
    Une fenêtre vous invite à créer l'activité correspondante à votre projet pour permettre la saisie des temps.
     
    Préciser le nom de l'activité Name/Nom, le Name detail/Nom long pour plus de précision, le Short name/Nom court qui sera affiché pour la saisie des temps. Par défaut le nom est celui du projet en cours.
     
    Sélectionner la Family/Famille de rattachement, une Sub family/Sous famille, si nécessaire.
     
    Si vous souhaitez ouvrir votre projet à la saisie, sélectionner Activated/Activé = Yes/Oui.
     
    L'ouverture à la saisie ou la fermeture à la saisie du projet peut être aussi activée dans le module activité/management.
     
     
     
     
     
    Lorsque que vous avez décidé de basculer votre projet dans l'état "In progress/En cours" il devient accessible dans le liste des projets en cours. L'onglet budget est enrichi des indicateurs d'avancement (consommé, forecast)
     
  • Passage de "In progress/En cours" vers "Opportunity/Opportunité"
     
    Vous pouvez basculer un projet d'un statut "In progress/En cours" à "Opportunity/Opportunité" si et seulement si il n'y a pas eu de saisie des temps sur le projet.
     
  • Passage de "In progress/En cours" ou  "Opportunity/Opportunité" vers "Archived/Archivé"
     
    Si votre projet est terminé ou si un projet en opportunité est arrêté, changer le statut, sélectionner l'onglet "Detail", changer le statut en Archived/Archivé puis, sauvegarder le projet.
     
  • Accéder à un projet Archived/archivé
 
Pour visualiser un projet archivé, sélectionnner Project/Projet puis dans le filtre statut, sélectionner Archived/archivé.
